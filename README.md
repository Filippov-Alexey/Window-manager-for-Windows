# Оконный менеджер для Windows (v2.0)

Этот проект представляет собой продвинутый оконный менеджер для Windows с модульной архитектурой. Система позволяет полностью заменить стандартную панель задач, управлять окнами и глубоко взаимодействовать с вводом данных.

## 🚀 Новое в обновлении

### 1. Улучшенная работа с модулями (Plugins)
Система стала более гибкой и автономной:
*   **Автоматический импорт:** Плагины теперь загружаются автоматически из папки `plugins`. Для успешного импорта должно соблюдаться условие: **имя папки == имя файла == имя класса**.
*   **Стандарт запуска:** Каждый класс плагина обязан содержать метод `run`, принимающий **4 аргумента**.
*   **Hot Reload:** Реализован автоматический перезапуск всех библиотек и плагинов при изменении файла конфигурации `variable.py`.

### 2. Расширенное логирование
*   Система логирования была оптимизирована для более детального отслеживания состояния сокетов, процессов импорта модулей и системных ошибок.

### 3. Глубокая работа с клавиатурой
Объекты событий клавиатуры (порт 65431) теперь включают дополнительные поля для точного анализа:
*   `key`: Символьное представление нажатой клавиши.
*   `isInjected`: Определяет тип нажатия (**физическое** или **программное**).
*   `layout`: Данные о текущей раскладке:
    ```json
    "layout": {"Name": "Russian", "HKL": "0x0000000004190419", "ID": "0x0419"}
    ```

### 4. Работа с буфером обмена
*   Добавлены инструменты для мониторинга и управления системным буфером обмена Windows.

## 🛠 Функционал

### Работа с клавиатурой
*   **keyboard_server.py**: Сокет-сервер (порт 65431), передающий нажатия клавиш в формате JSON.
*   **blocking.exe**: Блокировка клавиш и сочетаний (например, `blocking end left_ctrl+s`). Список кодов находится в `key_code.txt`.

### Работа с панелью задач
При запуске стандартная панель задач Windows скрывается, и активируются следующие компоненты:
*   **panel.py**: Управляет видимостью стандартного таскбара и запускает дочерние процессы.
*   **update_status.py**: Мониторинг системных показателей.
*   **update_icons.py**: Панель запущенных приложений (Taskbar/Dock).
*   **shortcut_panel.py**: Панель ярлыков для быстрого запуска.

### Работа с окнами
*   **update_grap.py**: Отрисовка графических элементов (например, обводка активного окна).
*   **windows_control.py**: Программное управление окнами.
*   **win.exe**: Модуль блокировки нежелательного раскрытия окон.
*   **getwin.exe**: Возвращает массив данных об окнах: `[(hwnd, title, path, rect), ...]`.

### Сетевая архитектура (Сокеты)


| Сервер | Порт | Макс. клиентов | Назначение |
| :--- | :--- | :--- | :--- |
| **Keyboard Server** | 65431 | 4 | Передача событий нажатия клавиш |
| **Windows Server** | 65432 | 3 | Список текущих окон (`getwin.exe`) |
| **Full Server** | 65433 | 4 | Статус полноэкранных приложений |

## 📦 Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com](https://github.com/Filippov-Alexey/Window-manager-for-Windows/)
   cd Window-manager-for-Windows```
2. **Установите зависимости:**
bash```
pip install -r requirements.txt```

3. **Запустите программу**:
bash```
python run.py```
