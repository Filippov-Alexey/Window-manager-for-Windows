# Оконный менеджер для Windows

Этот проект представляет собой оконный менеджер для Windows с расширенным функционалом, включающим работу с клавиатурой, панелью задач и окнами. Основные компоненты и функции описаны ниже.

## Функционал

### Работа с клавиатурой

- **keyboard_server.py**: Запускает сокет-сервер на порту 65431, поддерживающий до 4 клиентов. Сервер отправляет нажатые клавиши в виде кортежа. Например:
    ```json
    {
        "key_code": "0x20",
        "key_name": "space",
        "status": "Down",
        "pressed_keys": "",
        "option": "shift+ctrl+space, shift+left_ctrl+space, left_shift+ctrl+space, left_shift+left_ctrl+space",
        "blocked": "false"
    }
    ```
    и
    ```json
    {
        "key_code": "0x20",
        "key_name": "space",
        "status": "Up",
        "pressed_keys": "",
        "option": "shift+ctrl+space, shift+left_ctrl+space, left_shift+ctrl+space, left_shift+left_ctrl+space",
        "blocked": "false"
    }
    ```

- **blocking.exe**: Запускается с параметрами (например, `blocking end left_ctrl+s`) для блокировки клавиши и сочетания клавиш. Полный набор клавиш перечислен в файле `key_code.txt`.

### Работа с панелью задач

- При запуске программы стандартная панель задач скрывается, и создается собственная панель задач с доком открытых приложений и ярлыками.
- **panel.py**: Скрывает стандартную панель задач и запускает следующие компоненты приложения:
    - **update_status.py**: Мониторинг состояния системы.
    - **update_icons.py**: Отображение открытых программ.
    - **update_grap.py**: Простая обводка окон.
    - **shortcut_panel.py**: Отображение ярлыков на панели задач.

### Работа с окнами

- **update_grap.py**: Простая обводка окон.
- **windows_control.py**: Управление окнами.
- **win.exe**: Блокирует раскрытие окон.
- **getwin.exe**: Возвращает массив с множествами вида `[(hwnd, title, path, rect), ...]`.

### Сокет-серверы

- **keyboard_server.py**: Отправляет клиентам (до 4) нажатые клавиши по порту 65431.
- **windows_server.py**: Отправляет клиентам (до 4) результат выполнения программы `getwin.exe` по порту 65432.
- **windows_is_full_server.py**: Отправляет клиентам (до 4) нажатые клавиши по порту 65433.

### Сокет-клиенты

- **keyboard_manager.py**: Присоединяется к порту 65431.
- **shortcut_panel.py**: Присоединяется к порту 65433.
- **update_status.py**: Присоединяется к порту 65433.
- **update_grap.py**: Присоединяется к порту 65432.
- **update_icons.py**: Присоединяется к порту 65432.

### Дополнительно

- **hookdll.dll**: Подключается к `win.exe`.
- **vol.exe**: Возвращает уровень громкости.
- **run.py**: Запускает все необходимые скрипты.
- **variable.py**: Содержит основные константы.

## Установка и запуск

1. Склонируйте репозиторий на вашу локальную машину:
   ```bash
   git clone https://github.com/Filippov-Alexey/Window-manager-for-Windows.git
   cd Window-manager-for-Windows
2. Установите необходимые зависимости (если применимо).
   ```bash
   pip install -r requirements.txt

3. Запустите основную программу:

Bash

   python run.py
