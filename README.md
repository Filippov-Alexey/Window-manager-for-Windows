Window manager for windows - комплексная система кастомного рабочего окружения для Windows, заменяющая стандартный интерфейс (Taskbar, Alt-Tab, Window Manager) на производительное решение с тайлингом и динамическими плагинами.
## 🏗️ Архитектура системы
Проект построен на гибридной модели:

   1. Core (Python): Логика управления, оркестрация процессов, межисполняемое взаимодействие (IPC) и графический интерфейс (Tkinter).
   2. Services (C++ / Бинарные компоненты): Низкоуровневые хуки Windows API (CBT, Keyboard, Shell) для перехвата событий системы в реальном времени.

## Механизм взаимодействия (IPC)
Модули общаются через локальные TCP-сокеты (localhost).

* Протокол: JSON + сжатие zlib.
* Модель: Издатель-Подписчик (через manager.py) и прямой запрос-ответ.

------------------------------
## 🛠️ Основные компоненты## 1. Ядро (Kernel)

* run.py: Watchdog-процесс. Запускает виртуальное окружение, инициализирует модули и следит за их жизненным циклом. Перезапускает модули при изменении конфига (variable.py) и при изменении мониторов.
* manager.py: Центральный хаб данных об окнах. Позволяет любому плагину получать актуальную информацию о состоянии рабочего стола без прямого обращения к WinAPI.
* panel.py: Главная графическая оболочка на Tkinter, работающая в режиме topmost и заменяющая системный Taskbar.

## 2. Системные серверы

* windows_server.py (getwin.exe): Поставляет JSON-массив данных обо всех окнах (HWND, путь к EXE, иконка, координаты, статус активности).
* keyboard_server.py (blocking.exe): Глобальный хук клавиатуры. Читает нажатия и позволяет блокировать системные комбинации.
* space_server.py (space.exe): Реализует концепцию виртуальных рабочих столов через окна-якоря.
* windows_controle.py (win.exe): Перехватывает попытки изменения состояния окон (например, блокирует стандартную максимизацию для нужд тайлинга).

## 🧩 Плагины панели (Plugins)

| Плагин | Функционал |
|---|---|
| update_icons | Динамическая панель задач. Отображает иконки запущенных окон с кэшированием графики. |
| update_grap | Визуализация границ окон. Рисует градиентные рамки и бегущую строку заголовка активного окна. |
| update_status | Мониторинг ресурсов (CPU, GPU, RAM, Net, Vol) и индикатор/переключатель раскладки. |
| alt_tab | Кастомный переключатель окон с живыми превью (через PrintWindow). |
| shortcut_panel | Launchpad для быстрого запуска программ из .lnk файлов с проверкой на дубликаты запущенных копий. |

------------------------------
## 📐 Оконный менеджер (TWM)
Система реализует автоматическое управление окнами в нескольких режимах:

* Grid: Классическая сетка.
* Master: Одно главное окно спрва, остальные — в вертикальном стеке слева.
* BSP: Бинарное разбиение пространства (каждое новое окно делит текущую область пополам).

## ⚙️ Технические особенности

* Скрытие системного Taskbar: Реализовано программно через ShowWindow и поиск хэндла Shell_TrayWnd.
* Оптимизация: Использование numba для математических расчетов границ и threading для предотвращения фризов UI.
* Гибкость: Настройка через variable.py (маржины, порты, фильтры игнорируемых окон).
