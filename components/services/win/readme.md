# 🪟 Windows Window Tiling Manager & Hook Monitor

Интегрированный программный комплекс для низкоуровневого перехвата, логирования и принудительного тайлинга (управления геометрией) окон в ОС Windows.

---

## 📋 Сквозная Спецификация Кодов Операций (0 - 15)

Система использует единую беспрефиксную нумерацию для управления и отслеживания любых изменений состояний окон.

| Код (`code`) | Идентификатор (`name`) | Win32 Источник | Описание действия при блокировке (Передача аргумента) |
| :--- | :--- | :--- | :--- |
| **`0` – `11`**| `SW_HIDE` ... `SW_FORCEMINIMIZE` | `ShowWindow` / `HCBT_MINMAX` | Сворачивание, разворачивание, скрытие и восстановление окон по стандартной таблице Win32. |
| **`12`** | `CUSTOM_MOVE` | `CBTProc (HCBT_MOVESIZE)` | **Замораживает окна на месте (запрет ручного перемещения мышью за заголовок).** |
| **`13`** | `CUSTOM_SIZE` | `CBTProc (HCBT_MOVESIZE)` | **Запрещает ручное изменение геометрии (растягивание за края окон).** |
| **`14`** | `CUSTOM_CLOSE` | `CallWndProc (WM_CLOSE)` | **Блокирует закрытие окон (Крестик, Alt+F4, системное меню).** |
| **`15`** | `CUSTOM_SHOWWINDOW`| `CallWndProc (WM_SHOWWINDOW)`| Блокирует прямые посылы сообщений изменения видимости окон. |

---

## 🚀 Сборка Компонентов (C++)

Компиляция должна производиться строго в **Developer Command Prompt** для Visual Studio (архитектура x64).

### 1. Компиляция Хук-Библиотеки (`hookdll.cpp`)
```cmd
cl.exe /LD /O2 /EHsc hookdll.cpp /link /SECTION:.shared,RWS /OUT:hookdll.dll
```

### 2. Компиляция Сервера Ядра (`win.cpp`)
```cmd
cl.exe /O2 /EHsc /std:c++17 win.cpp user32.lib /link /OUT:win.exe
```

## 💻 Спецификация Интерфейса Данных (IPC)

Сервер `win.exe` транслирует события в `stdout` в унифицированном, строгом JSON-подобном формате с ключами в нижнем регистре на английском языке:

```json
{code:12, name:"CUSTOM_MOVE", title:"Администратор: Windows PowerShell", hwnd:263004}
{code:9, name:"SW_RESTORE", title:"*Безымянный – Блокнот", hwnd:524628}
```
